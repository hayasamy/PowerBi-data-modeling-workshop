📌 Overview

This repository contains files created as part of a Power BI data modeling workshop, completed during a professional training course by Pragmatic Works.

The project focuses on building an effective Power BI data model using best practices in dimensional modeling and star schema design, with an emphasis on scalability, performance, and real-world analytics use cases.

📂 Repository Contents

Source Data Files
Raw data used during the workshop for modeling and analysis.

Power BI File (.pbix)
The Power BI report containing:

Star schema data model

Fact and dimension tables

Relationships designed for analytical queries

🧠 Key Concepts Applied

Star schema & dimensional modeling fundamentals

Proper separation of fact and dimension tables

Avoiding common Power BI modeling mistakes

Designing models that scale with growing data

🎓 Course Context

This project was developed as part of a hands-on workshop in a Power BI data modeling course by Pragmatic Works.
The goal was to apply theoretical concepts in a practical, production-style Power BI model.

🚀 Purpose

Demonstrate practical Power BI data modeling skills

Showcase understanding of BI best practices for recruiters and hiring managers

Serve as a reference for future Power BI projects
